# 044-admin-console-redesign

InterCraft management console redesign.

## Summary

REQ-044 replaces the previous admin dashboard and log-center direction with a
new product-led management console. The console is designed primarily for PMs
who need product, AI-quality, and system-health signals for operating and
tuning InterCraft, while still giving operations and maintainer developers a
safe path from business anomaly to logs, traces, incidents, and audit evidence.
